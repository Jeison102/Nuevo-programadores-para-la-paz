Sistema operativo

Es el software principal del computador donde se ejecuta el servidor. Administra los recursos del equipo y permite que las aplicaciones funcionen correctamente.

Node.js

Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear aplicaciones y servidores web.

Aplicación server.js

Es el archivo que contiene el código del servidor. Define las rutas y las respuestas que enviará a los usuarios cuando hagan solicitudes.

Puerto

Es el número mediante el cual el servidor recibe las conexiones de los usuarios. En esta actividad se utiliza el puerto 3000.

Logs

Son registros que muestran la actividad del servidor y ayudan a identificar errores, monitorear el funcionamiento y realizar mantenimiento.
